# 数据结构课程实验仓库说明

本仓库用于发布实验代码与实验说明。  

## 1. 首次使用：克隆仓库

1. 打开终端（PowerShell / Git Bash / VS Code Terminal）。
2. 进入你想存放代码的位置。
3. 执行：

```bash
git clone https://github.com/KevinDb123/DSAProject.git
cd DSAProject
```

4. 检查是否成功：

```bash
git status
```

如果看到 `working tree clean`，说明仓库状态正常。

## 2.更新（如果仓库内代码有变动）

```bash
git pull
```

建议每次实验前先阅读对应实验目录下的 `README`，再开始改代码。
